# SustainHub – Project Report

## 1. Project Title

**SustainHub – A Sustainable Development Goals (SDGs) Awareness Platform**

---

## 2. Project Objective & Description

### Objective

The primary objective of **SustainHub** is to create an interactive web platform that raises awareness about the **United Nations Sustainable Development Goals (SDGs)**. The project aims to provide users with accessible information about global sustainability challenges while encouraging responsible environmental and social practices through an engaging user interface.

### Project Description

SustainHub is a responsive frontend web application developed using HTML, CSS, and JavaScript. The platform presents information related to the 17 Sustainable Development Goals in a visually appealing and easy-to-understand format. It focuses on delivering a seamless user experience with responsive layouts, intuitive navigation, and interactive design elements.

The project demonstrates modern frontend web development practices while serving as an educational platform that promotes sustainability awareness.

## 6. Conclusion & Summary

SustainHub successfully demonstrates the development of a responsive frontend web application focused on promoting awareness of the United Nations Sustainable Development Goals. Throughout the project, I gained practical experience in designing responsive web pages, implementing interactive features using JavaScript, organizing project files, and managing version control with Git and GitHub.

The project strengthened my understanding of frontend web development concepts, user interface design, and collaborative development practices. In future versions, I plan to extend SustainHub by integrating a Node.js backend, Firebase database, user authentication, and AI-powered features to provide a more personalized and interactive learning experience.
