# 🌱 SustainHub – Frontend

> A responsive frontend web application that promotes awareness of the United Nations Sustainable Development Goals (SDGs).

## 📖 Overview

SustainHub is a frontend web application developed using **HTML, CSS, and JavaScript**. The project is designed to spread awareness about the **17 United Nations Sustainable Development Goals (SDGs)** through an engaging, responsive, and user-friendly interface.

The application focuses on delivering an intuitive user experience while showcasing modern frontend development practices.

---

## ✨ Features

* 🌍 Informative sections about the Sustainable Development Goals (SDGs)
* 🎨 Modern and responsive user interface
* 📱 Mobile-friendly design
* ⚡ Interactive elements using JavaScript
* 🧭 Smooth and intuitive navigation
* 💻 Cross-browser compatible

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
